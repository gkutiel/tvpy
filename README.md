## Installation
```shell
> pip install tvpy
```

## Get an API Key
You need to get an API key from [TMDB](https://www.themoviedb.org/settings/api) and save it as `key.txt` in your working directory.

## Usage
```shell
> mkdir Carnival.Row 
> tvpy Carnival.Row 
```
